# 2C2P Demo 

## Requirements

- Maven
- Java
- [Configured .env file](../../README.md)

## How to run


1. Build the jar

```
mvn package
```

2. Run the packaged jar

```
java -cp target/sample-jar-with-dependencies.jar com.test2c2p.sample.Server
```

3. If you're using the html client, go to `localhost:4242` to see the demo.
